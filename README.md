## 📋 Tentang

Repo ini berisi **proposal presentasi** untuk menawarkan solusi digitalisasi ke pelaku UMKM. Tersedia dalam 3 format siap pakai:

| Format | File | Kegunaan |
|--------|------|----------|
| 🌐 HTML | `proposal.html` | Halaman web interaktif, dark theme, responsive |
| 📄 PDF | `proposal.pdf` | Kirim via WA/email, siap cetak (A3) |
| 🖥️ PPTX | `proposal.pptx` | Presentasi 10 slide untuk meeting client |

## 🛠️ Deploy

Repo ini dideploy via **Cloudflare Pages**:

1. Buka [dash.cloudflare.com/pages](https://dash.cloudflare.com/pages)
2. **Connect to Git** → pilih repo `digitalisasi-umkm`
3. Build settings: kosongkan (static site)
4. Deploy 🚀

Halaman `index.html` otomatis redirect ke `proposal.html`.
