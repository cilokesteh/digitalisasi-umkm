# 💼 Proposal — Digitalisasi UMKM

Proposal penawaran solusi digital untuk UMKM Indonesia.

## Yang Ditawarkan

| Paket | Harga | Include |
|-------|-------|---------|
| POS Basic | Rp 1,5jt | Kasir, stok, struk, laporan, export Excel |
| POS Enterprise | Rp 3,5jt | + IMEI/SN, restock/retur, split payment, kartu, WA send, terlaris, stock opname, website 5 halaman |
| POS Premium | Rp 6jt | + Full custom UI, fitur spesial, integrasi, multi-cabang |
| Bundle (Enterprise + Website) | Rp 4,5jt | Hemat Rp 1,5jt dari beli terpisah |

## Tech Stack

- **POS:** HTML/CSS/JS + Firebase (Auth + Firestore) — PWA installable
- **Website:** Static HTML + Tailwind CSS → Cloudflare Pages
- **Domain:** Gratis 1 tahun (.com)

## File

- `proposal.html` — Landing page proposal (buka langsung di browser)
- `proposal.pdf` — Versi PDF
- `proposal.pptx` — Versi PowerPoint

## Buka Proposal

```bash
# Local
open proposal.html

# Atau deploy ke GitHub Pages / Cloudflare Pages
```
